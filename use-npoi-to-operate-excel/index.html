Use NPOI to Operate Excel

Mar 9th, 2014 | 评论

NPOI来操作Excel(现在也可以操作Word)非常方便,现在NPOI2已经出了,可以支持office2007以后的版本.

按照官方推荐的方式,试了一下自动识别Excel版本的功能,首先简要介绍一下:

NPOI2的主要部件

NPOI 2.0有几个DLL构成，他们分别是NPOI.DLL，NPOI.OOXML.DLL, NPOI.OpenXml4Net.dll, NPOI.OpenXmlFormats.dll，其中后三个是2.0新加的库。

NPOI.DLL中包含的模块

NPOI.Util   基础辅助库
  NPOI.POIFS   OLE2格式读写库，主要负责处理DocumentInformation   NPOI.DDF   Microsoft Drawing格式读写库  

NPOI.SS   Excel 2003和Excel 2007操作库共用的接口库以及公用模块实现，如公式计算库
   NPOI.HPSF   OLE2的Summary Information和Document Summary Information属性读写库
  NPOI.HSSF   Excel BIFF格式读写库，这是用户用的最多的命名空间
  NPOI.OOXML中包含的模块

NPOI.XSSF Excel 2007操作库，大部分对象都实现了NPOI.SS的接口 

NPOI.XWPF Word 2007操作库   NPOI.OpenXml4Net.DLL中包含的模块

只有一个，即NPOI.OpenXml4Net，它是从POI的子项目OpenXml4j移植过来的，其功能类似于NPOI.POIFS，只是它操作的是OOXML格式。这个模块和微软提供的System.Packaging功能是一致的，只是System.Packaging支持.Net 3.0以上，这个支持.NET 2.0。   NPOI.OpenXmlFormats.DLL中包含的模块 也只有一个，即OpenXmlFormats，它定义了所有OOXML对象和C#对象的映射关系，并协助进行序列化和反序列化，从而使文件读写层和逻辑层分离。

引用:http://tonyqus.sinaapp.com/archives/575

读取Excel文件

直接上代码,很简单

string filepath = string.Empty;

if (ofd_excel.ShowDialog() == System.Windows.Forms.DialogResult.OK)
{
    filepath = ofd_excel.FileName;
    tb_filepath.Text = filepath;
}

FileStream fs = File.OpenRead(filepath);
IWorkbook workbook = WorkbookFactory.Create(fs);

//show them all
ISheet sheet = workbook.GetSheet("Sheet1");

int sindex = 3;
string[] slist = new string[7];

while (true)
{
    if (sheet.GetRow(sindex).GetCell(0).ToString().Length < 1)
        break;

    for (int i = 0; i < 7; i++)
    {
        slist[i] = sheet.GetRow(sindex).GetCell(i).ToString();
    }

    ListViewItem lvi = new ListViewItem(slist);
    lv_main.Items.Add(lvi);
    sindex++;
}
